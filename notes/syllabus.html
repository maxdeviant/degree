Description This course teaches data structures with a strong focus on algorithm analysis. The Java programming language is used. One of the goals of the course is to study and understand the classes and operations which make up the Java collections. Several course assignments require Java programming and the student is expected to use a modern IDE such as NetBeans for programming development.
Content The course content relies on the instructor's notes and closely follows the development of the material in the textbook. Topics include:
  • Sections 1.1 – 1.3 (math review), 2.1 – 2.4, 3.1 – 3.7
    • Algorithm analysis using the order language: O, Ω, etc.
    • Lists, Deques, Stacks, Queues via array-based and link-based implemetations.
    • Stack-based algorithms for expression evaluation.
  • Sections 4.1 – 4.4, 5.1 – 5.6
    • Binary Trees, TreeSets, TreeMaps.
    • Hashing via open and closed implementations.
    • AVL Trees.
  • Sections 7.1 – 7.4, 7.6 – 7.9, 6.1 – 6.3, 7.5
    • Sorting algorithms and timing comparsions.
    • Priority Queues and Binary Heaps.

Integrity		 All work submitted for grade should be completed through your individual work alone.

You must to adhere to the university's standards for academic integrity. I will be scrutinizing your assignments looking for signs of plagiarism and deduct points as I see fit. For me it usually stands out "like a sore thumb." In many cases all parties involved will lose points unless I can ascertain who was the author.

I can guarantee that if you work alone, it will never be mistaken as the work of others, and so if you are in doubt about what to do, avoid asking for help from other students, or, on the other side of the coin, avoid revealing your work to others. If you cannot abide by these stipulations, please drop the course! I will not "pass you through" this course because you somehow think you are entitled to it!
