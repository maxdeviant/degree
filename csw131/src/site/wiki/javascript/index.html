Info

JavaScript is a dynamic programming language for the web. While originally only executed in the browser, recent developments in Node.js have made JavaScript code able to be executed on the server as well. This has made JavaScript become very popular, since the same programming language can be used for both the server and the client-side code in an application.

Best Practices

Due to its untyped and dynamic nature, coding in JavaScript can be very unstructured when compared to statically typed languages such as C or C++. Because of this, large JavaScript projects can get out of hand very quickly. That is why it is important to adhere to a number of best practices when working with JavaScript.
